## ⚡ أوامر سريعة

```bash
# التشغيل المحلي
pnpm dev

# البناء للإنتاج
pnpm build

# معاينة البناء
pnpm preview

# فحص الأخطاء
pnpm lint

# فحص أنواع TypeScript
pnpm typecheck
```
