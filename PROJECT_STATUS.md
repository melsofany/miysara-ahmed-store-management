## 🚀 الخطوات التالية للمستخدم

### خيار 1: البدء السريع (5 دقائق)

```bash
# 1. اقرأ START_HERE.md
# 2. اجمع المعلومات الأربع
# 3. شغّل السكريبت
export RENDER_API_KEY="..."
export RENDER_SERVICE_ID="..."
export VITE_SUPABASE_URL="..."
export VITE_SUPABASE_ANON_KEY="..."

node scripts/set-render-env.js

# 4. انقر Redeploy على Render
# 5. انتظر 2-5 دقائق
```

### خيار 2: استخدام Dashboard (10 دقائق)

```
1. Render Dashboard
2. Settings → Environment
3. أضف المتغيرات يدويّاً
4. Redeploy
5. انتظر وتحقق
```

### خيار 3: التطوير المحلي

```bash
# 1. استنساخ المشروع
git clone https://github.com/melsofany/miysara-ahmed-store-management.git
cd miysara-ahmed-store-management

# 2. التثبيت
pnpm install

# 3. إعداد البيئة
cp .env.example .env.development.local
# أضف بيانات Supabase

# 4. التشغيل
pnpm dev
```
