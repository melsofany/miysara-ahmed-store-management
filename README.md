# نظام إدارة سلسلة محلات الملابس - MIYSARA Ahmed

نظام متكامل وشامل لإدارة محلات الملابس مع دعم كامل للمخازن المتعددة، نقاط البيع، الفواتير، المخزون، التقارير والمزيد.

## الميزات الرئيسية

- 🏢 **إدارة متعددة المخازن** - دعم كامل لإدارة عدة مخازن وفروع
- 🛒 **نقاط بيع غير محدودة** - إنشاء وإدارة عدد غير محدود من نقاط البيع
- 📊 **المخزون والحركات** - تتبع شامل لحركة المخزون والتنبيهات التلقائية
- 💳 **الفواتير والمرتجعات** - نظام فواتير متقدم مع دعم المرتجعات الكاملة
- 📈 **التقارير والتحليلات** - تقارير مفصلة عن المبيعات والأرباح والمخزون
- 👥 **إدارة المستخدمين والأدوار** - نظام صلاحيات متقدم مع أدوار قابلة للتخصيص
- ⏱️ **إدارة الشفتات والدرج** - تتبع الشفتات وحركات الدرج بشكل دقيق
- 🔐 **سجل التدقيق الكامل** - تسجيل شامل لجميع العمليات والتغييرات

## المتطلبات

- Node.js 18+ أو pnpm 9+
- حساب Supabase
- متصفح حديث

## التثبيت

### 1. استنساخ المشروع

```bash
git clone https://github.com/melsofany/miysara-ahmed-store-management.git
cd miysara-ahmed-store-management
```

### 2. تثبيت الاعتماديات

```bash
pnpm install
```

### 3. إعداد متغيرات البيئة

انسخ ملف `.env.example` إلى `.env.development.local`:

```bash
cp .env.example .env.development.local
```

ثم أضف بيانات Supabase الخاصة بك:

```env
VITE_SUPABASE_URL=https://your-project.supabase.co
VITE_SUPABASE_ANON_KEY=your-anon-key-here
```

### 4. تشغيل المشروع

```bash
pnpm dev
```

سيتم فتح التطبيق على `http://localhost:5173`

## البناء والنشر

### البناء للإنتاج

```bash
pnpm build
```

### النشر على Render

المشروع مجهز بالفعل للنشر على Render. تأكد من:

1. إضافة متغيرات البيئة في لوحة تحكم Render:
   - `VITE_SUPABASE_URL`
   - `VITE_SUPABASE_ANON_KEY`

2. التأكد من أن `render.yaml` يستخدم الأوامر الصحيحة

## المصادقة

التطبيق يستخدم Supabase للمصادقة. يمكنك:

- إنشاء حساب جديد
- تسجيل الدخول بالبريد الإلكتروني وكلمة المرور
- إدارة الأدوار والصلاحيات

## الأدوار والصلاحيات

النظام يدعم أدوار متعددة:

- **مدير عام (Super Admin)** - صلاحيات كاملة
- **مدير النظام** - إدارة المستخدمين والإعدادات
- **مدير المخزن** - إدارة المخزون والحركات
- **مستخدم مخزن** - عرض وتسجيل الحركات
- **مدير نقطة البيع** - إدارة نقطة البيع
- **كاشير** - استخدام واجهة نقطة البيع
